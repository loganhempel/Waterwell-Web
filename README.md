# Waterwell-Web

Landing-page redesign for **Waterwell Clinic** — naturopathy & skin health, Auckland (practitioner: Kohei Iguchi).

A single self-contained marketing site: root-cause skin treatment (eczema, psoriasis, acne, topical steroid withdrawal), built to feel calm, premium and reassuring.

## Highlights
- **"Inside-Out" scroll schematic** — an animated clinical cross-section that draws itself from the skin surface down to the internal systems, illustrating the root-cause philosophy.
- Drag-to-compare and hover-reveal **before/after** patient results.
- FAQ accordion, animated stat counters, sticky mobile CTA, scroll progress.
- Type system: Fraunces · Hanken Grotesk · IBM Plex Mono. Brand palette: "Deep Current" navy `#0E3A64` + warm cream.

## Stack
Plain HTML / CSS / vanilla JS — no build step. Open `index.html` directly, or deploy the folder to any static host (e.g. Vercel).

```
index.html      # the site
assets/         # logo, photos, before/after images
```
